# GymTrackerApp 💪

A cross-platform mobile application built with **.NET MAUI** for tracking gym exercises and workout progress.

## Overview

GymTrackerApp allows users to manage their gym exercises by adding, editing, and deleting exercise entries. Exercise data is persisted locally on the device using JSON file storage.

## Features

- 📋 **View Exercises** – Browse all saved exercises on the main page
- ➕ **Add Exercise** – Log a new exercise with name, reps/time, and weight
- ✏️ **Edit Exercise** – Update existing exercise details
- 🗑️ **Delete Exercise** – Remove exercises with confirmation prompt
- 🔍 **Filter Exercise** – Filter exercises by type

## Tech Stack

- **.NET 9 / .NET MAUI** – Cross-platform UI framework
- **C# / XAML** – Application logic and UI
- **CommunityToolkit.Mvvm** – MVVM helpers
- **System.Text.Json** – Local JSON data storage
